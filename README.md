# Task Manager Web

Um gerenciador de tarefas simples e moderno desenvolvido com **HTML, CSS e JavaScript puro**, criado como parte do meu portfólio de estudos em desenvolvimento web.

## Funcionalidades

- Adicionar novas tarefas
- Remover tarefas individualmente
- Marcar tarefas como concluídas
- Salvar tarefas automaticamente no navegador com **localStorage**
- Manter tarefas concluídas salvas mesmo após atualizar a página
- Contador dinâmico de tarefas
- Botão para limpar todas as tarefas
- Interface responsiva e organizada
- Scroll funcional para listas grandes

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API
