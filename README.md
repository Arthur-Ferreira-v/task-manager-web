# Task Manager Web App

Aplicação web de gerenciamento de tarefas desenvolvida com HTML, CSS e JavaScript puro.

O projeto foi criado com foco em prática de front-end, manipulação do DOM e desenvolvimento de interfaces interativas.

---

# 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- DOM
- Flexbox

---

# ✨ Funcionalidades

✅ Adicionar tarefas  
✅ Remover tarefas  
✅ Marcar tarefas como concluídas  
✅ Adicionar tarefas usando Enter  
✅ Validação de input vazio  
✅ Interface moderna  
✅ Hover animations  
✅ Layout responsivo básico
